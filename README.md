## How to start
- Rebuild JAR (Ensure JAVA_HOME is set to JDK 23): 

```bash
./mvnw clean package -DskipTests 
```
- Run Docker:

```bash
docker-compose up --build -d
```
## API Documentation
- Swagger UI: http://localhost:8080/swagger-ui.html
- OpenAPI JSON: http://localhost:8080/v3/api-docs

## AI Collaboration Narrative
Strategy: I approached this as a "Pair Programming" session. I define the folder structure and service boundaries first. Start with write detail document about the implementation approach the let AI process my idea.

Verification & Refinement: Every module generated by the AI was immediately followed by a request to "Write JUnit tests for this logic." first before actual implement the function (TDD approach). When the AI initially used a deprecated library for the database connection, I manually intervened to update it to the latest Spring Boot standard and re-prompted for the updated configuration.

Quality Assurance: I conducted a final manual code review to ensure the implementation meet my expectation. Then process further action for every output from AI agent.
